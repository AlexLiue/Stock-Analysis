# RSI-相对强弱指数 (Relative Strength Index) 

> **文档信息**
> - **指标名称**: RSI (Relative Strength Index)
> - **创始人**: J. Welles Wilder Jr.

## 1. 什么是 RSI？
**RSI (相对强弱指数)** 是一种动量振荡指标，测量近期价格变动的速度和幅度，以评估资产价格的**超买**或**超卖**情况。其数值范围在 `0` 到 `100` 之间。

## 2. 核心数学模型
RSI 的计算公式如下：

$$RSI = 100 - \left[ \frac{100}{1 + \frac{\text{Average Gain}}{\text{Average Loss}}} \right]$$

*注：通常取 14 个周期（天、小时或分钟）作为标准计算长度。*  
*Average Gain：周期平均上涨幅度, 通常取 14 个周期（天、小时或分钟）作为标准计算长度。*  
*Average Loss：周期平均下跌幅度, 通常取 14 个周期（天、小时或分钟）作为标准计算长度。*  


### 2.1 详细计算步骤 (以 14 周期为例)

#### 第一阶段：获取原始变动
计算连续 15 个周期的收盘价。计算当日收盘价与前一日收盘价的差值（共 14 个差值）：
- 如果差值 > 0，记为 **Gain (上涨)**
- 如果差值 < 0，记为 **Loss (下跌，取绝对值)**
- 如果差值 = 0，则两者都记为 0

#### 第二阶段：计算初始平均值 (Initial Average)
对于第 14 个周期，使用简单平均法：
- **初始平均上涨** = (过去 14 天上涨点数之和) / 14
- **初始平均下跌** = (过去 14 天下跌点数之和) / 14

#### 第三阶段：计算平滑平均值 (Wilder 平滑法)
为了使 RSI 指标更加稳定，Wilder 采用了类似指数移动平均的平滑方式。从第 15 个周期开始：
1. **当前平均上涨** = `[(前一日平均上涨 × 13) + 当日上涨幅度] / 14`
2. **当前平均下跌** = `[(前一日平均下跌 × 13) + 当日下跌幅度] / 14`

#### 第四阶段：计算RSI . 实例解析
假设连续 14 天的平均上涨为 2.5 点，平均下跌为 1.25 点：

1. **计算 RS**:
   $$RS = 2.5 / 1.25 = 2.0$$
2. **计算 RSI**:
   $$RSI = 100 - (100 / (1 + 2))$$
   $$RSI = 100 - 33.33 = 66.67$$

**结论**: 该资产目前处于相对强势状态（接近 70 的超买警戒线）。


## 3. 关键数值区间与含义
| RSI 数值 | 市场含义 | 交易建议 |
| :--- | :--- | :--- |
| **> 70** | **超买 (Overbought)** | 警惕回调风险，考虑减仓或分批止盈 |
| **< 30** | **超卖 (Oversold)** | 价格可能见底，关注反弹买入机会 |
| **50** | **多空分界线** | 上方为强热市场，下方为弱势市场 |

## 4. 核心交易策略
### 4.1 背离信号 (Divergence) —— 最强反转预警
- **顶背离**：价格创新高，但 RSI 走低。预示趋势即将见顶转跌。
- **底背离**：价格创新低，但 RSI 走高。预示趋势即将见底回升。

### 4.2 RSI 趋势线
你可以直接在 RSI 指标线上画出支撑和压力线。RSI 对趋势线的突破通常比价格图表更早发生，具有**领先性**。

### 4.3 极端值说明
- **RSI = 100**: 在统计周期内，价格**每日都在上涨**，AvgLoss 为 0。
- **RSI = 0**: 在统计周期内，价格**每日都在下跌**，AvgGain 为 0。
- **RSI = 50**: 上涨和下跌的力度完全对等。

### 5.2 Python 实现 
```python
def calculate_rsi(data, window=14):
    delta = data.diff()
    gain = (delta.where(delta > 0, 0))
    loss = (-delta.where(delta < 0, 0))

    # 使用 Wilder 平滑法 (EWM)
    avg_gain = gain.ewm(com=window-1, min_periods=window).mean()
    avg_loss = loss.ewm(com=window-1, min_periods=window).mean()

    rs = avg_gain / avg_loss
    rsi = 100 - (100 / (1 + rs))
    return rsi
```
